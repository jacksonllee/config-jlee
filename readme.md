Building a Python distribution from Source on Ubuntu
----------------------------------------------------

Get the `.tar.xz` from python.org.
Then unpack and install the Python distribution:

```
tar xf Python-x.x.x.tar.xz
cd Python-x.x.x
./configure
make
make test
sudo make install
```
